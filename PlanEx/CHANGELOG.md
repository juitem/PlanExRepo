# Changelog

## [1.0.0] - 2026-03-13

### Added
- Initial release of PlanEx.
- Folder comparison via `PlanEx: Compare Folders` command.
- Sidebar TreeView displaying hierarchical diff (Added/Deleted/Modified/Unchanged).
- Click-to-diff for modified files using VS Code's built-in Diff Editor.
- Merge file, merge folder, and delete-from-target inline actions.
- Next/Previous changed file navigation in the tree view.
- Symlink loop protection during directory traversal.
- Architecture Decision Records (ADR) documented in `docs/adr/`.
