# JOB_HISTORY — Job 진행 이력

> **목적**: 각 Job의 시작, 완료, 실패, 재시도 이력을 기록한다. 삭제 금지.
> **갱신 규칙**: Job 시작, Phase 완료, Job 완료, 실패 발생 시 항목 추가

---

## 이력 기록 형식

```markdown
### [YYYY-MM-DD] JOB-XXX [이벤트 유형]

| 항목 | 내용 |
|------|------|
| 날짜 | YYYY-MM-DD |
| 에이전트 | [역할] |
| Job | JOB-XXX |
| 이벤트 | STARTED / PHASE_COMPLETED / COMPLETED / FAILED / RETRYING |
| Phase | [Phase X-X] (Phase 이벤트일 때) |

**내용**
[이벤트 상세 설명]

**산출물** (완료/Phase완료 시)
- [산출물 목록]

**실패 원인** (실패 시)
- [실패 원인]

**재시도 전략** (재시도 시)
- [전략]
```
